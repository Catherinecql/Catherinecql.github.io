Projects

FiB

A chrome-extension goes through your facebook feed in real time as you browse it and verifies the authenticity of posts. The posts are visually tagged on the top right corner in accordance with their trust score. If a post is found to be false, the AI tries to find the truth and shows it to you. Each time a user posts/shares content, the user will be notified if the new post contains any unverified information. If so,and can choose to either take it down or let it exist.

LiveMessage Dispatcher

A full stack project by creating a Heroku app which connects to Salesforce Dev Org and to JavaClient. It enables end to end test on RealTime LiveMessage flow to individual developer machines with real and live data, which shortened the code testing cycle from days to seconds.
